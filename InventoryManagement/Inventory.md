# Inventory Management
## Instruction
1. Install Apache, PHP, and MySQL
```bash
sudo apt update
sudo apt install apache2 php libapache2-mod-php php-mysql mysql-server -y
``` 
2. Start Apache and MySQL
```bash
sudo service apache2 start
sudo service mysql start
```
3. Then, copy code into `/var/www/html`
```bash
sudo cp -r /path/to/project /var/www/html
```
4. In your web browser, go to http://localhost/
5. Click into mainmenu.php
